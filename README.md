# dotfiles

## Usage

dotfiles managed by [chezmoi](https://www.chezmoi.io/).
```shell
chezmoi init --ssh jessun
```
or
```shell
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply --verbose https://github.com/jessun/dotfiles.git
```
